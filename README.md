## 🚀 安裝與使用指南 (Installation & Usage)

1. **取得檔案**：
   下載本專案提供之 `TwsausageApp.7z` 壓縮檔。
2. **解壓縮**：
   請準備 [7-Zip](https://www.7-zip.org/) 或支援 `.7z` 格式的解壓縮軟體。對檔案點擊右鍵並選擇「解壓縮至此」。
3. **執行程式**：
   進入解壓縮後的資料夾，找到主執行檔（如 `TwsausageApp.exe`），雙擊即可開啟應用程式。
